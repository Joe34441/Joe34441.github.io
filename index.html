Wave Function Collapse [WIP]

An implementation of the Wave Function Collapse algorithm to generate a 3d city-like level. Made in Untiy with c#, I implemented a grid-based tiled model with the algorithm consisting of cell selection through entropy evaluation, cell collapse through weighting and propagation via arc consistency.
Available at: <git repo>

Water Shader

A water shader made in Unity with the Universal Render Pipeline. I implemented the trochoidal/gerstner algorithm for the waves, with multiple other features in shader graph to produce a realistic water visual. There are also accompanying effects created with HLSL and the VFX graph to create the ocean storm diorama.
Available at: <git repo>
